### Synopsis

FormattedHTMLMessage React component

### Description

Does the same as FormatedMessage, but should be used for rendering 'unsafe' text
that could cause potentially XSS problems.

Read more [Dangerously Set innerHTML](http://reactjs.cn/react/tips/dangerously-set-inner-html.html)

### Code Example

```
<FormattedHTMLMessage message="logo"/>
```

### Props Reference

| Name    | Type   | Description     |
| ------- |:-------| ----------------|
| message | string | translation key |
